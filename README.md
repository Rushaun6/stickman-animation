# Stickman Animation

A Python-based animation engine for creating stickman animations and movies.

## Project Structure

- `stickmovie/` - Main package directory
  - `main.py` - Entry point
  - `engine.py` - Core animation engine
  - `renderer.py` - Rendering system
  - `characters.py` - Character definitions
  - `animations.py` - Animation sequences
  - `backgrounds.py` - Background management
  - `dialogue.py` - Dialogue system
  - `effects.py` - Visual effects
  - `scenes.py` - Scene management
